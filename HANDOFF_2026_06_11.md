## 4. The DSSP recipe (#15 — hard-won, do NOT re-derive)

DSSP 4.x (`mkdssp`, libcifpp-based) on the Modal image could not read the minimal ESMFold2 CIFs — they omit the mmCIF polymer categories cifpp builds residues from, so it saw **0 residues**. Fix, in `surface_analysis.py:compute_secondary_structure`:

1. Load the structure via the tolerant `cif_io` loader.
2. Write it to a **temp PDB**, and **prepend a `HEADER` record** — cifpp sniffs PDB-vs-mmCIF by the *first line*; without `HEADER` it assumes mmCIF and fails.
3. Run `mkdssp --output-format dssp <pdb>` — **force the legacy format**; DSSP 4.x defaults to mmCIF output, which BioPython's parser reads as empty.
4. Parse with `make_dssp_dict`. **Gate `dssp_success` on actual assignments**, never on a clean exit — a silent 0-residue result must not read as a reliable measurement (that bug masqueraded as "reliable: true, 0 assignments").

Validated: 6EQE 35/20 H/E → α/β hydrolase; 1UBQ 25/34 → β-grasp; and all five demos give real, reliable SS.
