# Globular enzyme

Expected parameters for a compact, single-domain globular enzyme. Bounds are
**calibrate-on-real-data placeholders**.

| parameter | min | max | unit | note |
|---|---|---|---|---|
| asphericity |  | 0.30 |  | > 0.30 → elongated, atypical for a compact globular domain |
| buried_fraction | 0.30 |  |  | folded hydrophobic core present |
| coil_fraction |  | 0.45 |  | high coil → poor packing or disorder |
| sheet_fraction | 0.05 |  |  | most α/β and β enzymes carry some sheet |
