# De novo monomer

Expected parameters for an idealised de novo designed monomer — typically
highly ordered, compact, and roughly charge-balanced. Bounds are
**calibrate-on-real-data placeholders**.

| parameter | min | max | unit | note |
|---|---|---|---|---|
| coil_fraction |  | 0.35 |  | de novo designs are usually highly ordered |
| buried_fraction | 0.35 |  |  | designed compact core |
| asphericity |  | 0.35 |  | most designs compact; elongated bundles are the exception |
| surface_net_charge | -12 | 12 | e | designs are often roughly charge-balanced |
